## Running Locally


```sh-session
git clone https://github.com/jclark/jclark.io.git
cd jclark.io
```


Create a `.env` file similar to [`.env.example`](https://github.com/jclark/jclark.io/blob/main/.env.example).

Then install dependencies and run the development server:
```sh-session
pnpm install
pnpm dev
```
